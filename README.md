# AA-Alpha-Bruno
Annuity Association Alpha Bruno Config

## Setup Instructions

### Bruno Setup
1. Install Bruno Desktop App from [Bruno's official website](https://www.usebruno.com/downloads)
2. Clone this repository to your local machine
3. Open Bruno Desktop App
4. Click "Open Collection" and navigate to the cloned repository folder
5. Select the folder containing the `bruno.json` file

## Usage
This collection contains API configurations for the Annuity Association Alpha project. Use Bruno to send requests and test endpoints defined in the `.bru` files.
